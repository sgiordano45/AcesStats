üìä Preview Fields

7 WhatsApp Sharing

Several content types can be shared directly to WhatsApp with pre-formatted messages.

üì± What Gets Shared

Eulogies

  • Team name with eulogy/celebration title
  • Full eulogy text
  • Link to the Year in Review page

Game Previews

  • Matchup (Away @ Home)
  • Game date and time
  • Peters Odds (if entered)
  • Peters Preview text (if entered)
  • Link to Weekend Preview page
üí° How It Works
Clicking the WhatsApp button opens WhatsApp with a pre-formatted message. You can then choose which chat or group to send it to. The content uses whatever is currently in the form fields, so make sure to fill them in first!
