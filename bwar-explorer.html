⚾ bWAR Explorer BETA

Calibration tool — tune batting WAR weights before publishing to player pages

How this works: bWAR = Batting Wins Above Replacement. Each player's production is measured by wOBA (weighted on-base average, using configurable run values), compared to a replacement-level baseline (% of league average), scaled to runs and then converted to wins using the league's actual run environment.

Full data — player has 2B/3B/HR tracked (game tracker or PDF import).
Imputed — only H/AB/BB known; 2B/3B/HR estimated from league-average XBH rates.
Simplified — all hits treated equally (flat weight); shown alongside for comparison.

Adjust the sliders and click Recalculate (or it updates live). Save your preferred weights at end of season once you're happy with how the rankings look.
